# CIDR Notation

CIDR = Classless Inter-Domain Routing

Format:

192.168.1.0/24

Where:

- Network Address = 192.168.1.0
- Prefix Length = 24

## Common CIDR Blocks

| CIDR | Total IPs |
| ---- | --------- |
| /32  | 1         |
| /30  | 4         |
| /29  | 8         |
| /28  | 16        |
| /27  | 32        |
| /26  | 64        |
| /25  | 128       |
| /24  | 256       |
| /16  | 65536     |

## Example

VPC CIDR:

10.0.0.0/16

Subnet CIDR:

10.0.1.0/24

10.0.2.0/24
