# Subnetting

Subnetting is the process of dividing a larger network into smaller networks.

These smaller networks are called subnets.

---

## Why Subnetting?

Benefits:

- Better organization
- Better security
- Better performance
- Reduced broadcast traffic

---

## Example

Network:

10.0.0.0/16

Subnets:

10.0.1.0/24

10.0.2.0/24

10.0.3.0/24

---

## AWS Example

VPC:

10.0.0.0/16

Public Subnet:

10.0.1.0/24

Private Subnet:

10.0.2.0/24

---

## Advantages

### Security

Separate web servers and databases.

### Management

Easier administration.

### Scalability

Add more subnets when needed.
