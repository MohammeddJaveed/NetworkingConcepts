# TCP/IP Model

The TCP/IP model is the practical networking model used on the Internet.

Unlike OSI's 7 layers, TCP/IP uses 4 layers.

---

## TCP/IP Layers

| TCP/IP Layer   | OSI Mapping  |
| -------------- | ------------ |
| Application    | Layers 5,6,7 |
| Transport      | Layer 4      |
| Internet       | Layer 3      |
| Network Access | Layers 1,2   |

---

## Application Layer

Combines:

- Application
- Presentation
- Session

Protocols:

- HTTP
- HTTPS
- DNS
- SMTP
- FTP

---

## Transport Layer

Responsible for communication between devices.

Protocols:

### TCP

Reliable

Features:

- Acknowledgement
- Retransmission
- Sequencing

### UDP

Fast

Features:

- No acknowledgement
- No retransmission

---

## Internet Layer

Responsible for routing.

Protocols:

- IPv4
- IPv6
- ICMP

---

## Network Access Layer

Responsible for:

- Physical transmission
- MAC addressing

Examples:

- Ethernet
- WiFi

---

## OSI vs TCP/IP

| OSI            | TCP/IP           |
| -------------- | ---------------- |
| 7 Layers       | 4 Layers         |
| Theoretical    | Practical        |
| Teaching Model | Real-world Model |
