# OSI Model

OSI stands for Open Systems Interconnection.

It contains 7 layers.

| Layer Number | Layer Name   | Function                |
| ------------ | ------------ | ----------------------- |
| 7            | Application  | User interaction        |
| 6            | Presentation | Encryption, Compression |
| 5            | Session      | Session management      |
| 4            | Transport    | Reliable communication  |
| 3            | Network      | Routing                 |
| 2            | Data Link    | MAC addressing          |
| 1            | Physical     | Transmission of bits    |

## Layer 7 - Application

Provides services directly to users.

Examples:

- HTTP
- HTTPS
- FTP
- SMTP

## Layer 6 - Presentation

Responsible for:

- Encryption
- Compression
- Data formatting

Examples:

- SSL
- TLS

## Layer 5 - Session

Responsible for:

- Session creation
- Session maintenance
- Session termination

## Layer 4 - Transport

Responsible for:

- End-to-end communication
- Reliability

Protocols:

- TCP
- UDP

## Layer 3 - Network

Responsible for:

- Routing packets

Protocols:

- IP
- ICMP

Devices:

- Router

## Layer 2 - Data Link

Responsible for:

- MAC Addressing
- Frame delivery

Devices:

- Switch

## Layer 1 - Physical

Responsible for:

- Bit transmission

Examples:

- Cable
- Fiber
- Wireless signals

## Interview Question

Q: Which OSI layer does TCP work on?

Answer:
Layer 4 (Transport Layer).
