# VPC

VPC = Virtual Private Cloud

A logically isolated network in the cloud.

Think of VPC as your private data center inside AWS.

## Example

VPC CIDR:

10.0.0.0/16

Available IPs:
65,536

## Components

- VPC
- Subnets
- Route Tables
- Internet Gateway
- NAT Gateway
- Security Groups
- NACLs

## Benefits

- Isolation
- Security
- Scalability
- Control over networking
