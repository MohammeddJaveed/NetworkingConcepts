# Networking Interview Questions

This file contains common networking questions with concise answers.

## Basic Questions

### What is a network?

A network is a group of connected devices that can exchange data.

### What is the difference between a switch and a router?

A switch connects devices within the same local network. A router connects different networks and forwards packets between them.

### What is an IP address?

An IP address is a logical address used to identify a device or interface on an IP network.

### What is a MAC address?

A MAC address is a hardware address used for local network communication at the data link layer.

### What is a subnet?

A subnet is a smaller network created from a larger IP network.

### What is CIDR?

CIDR is a notation for describing network size, such as `192.168.1.0/24`.

## Protocol Questions

### What is TCP?

TCP is a reliable, connection-oriented transport protocol that provides ordered delivery and retransmission.

### What is UDP?

UDP is a lightweight, connectionless transport protocol without built-in reliability or ordering.

### What is DNS?

DNS translates domain names into IP addresses.

### What is DHCP?

DHCP automatically assigns IP configuration such as IP address, subnet mask, gateway, and DNS servers.

## Cloud Networking Questions

### What is a VPC?

A VPC is a logically isolated private network inside a cloud provider.

### What makes a subnet public?

A subnet is public when its route table has a route to an internet gateway.

### What is a NAT gateway?

A NAT gateway allows private subnet resources to initiate outbound internet connections without being directly reachable from the internet.

### What is a security group?

A security group is a stateful firewall attached to cloud resources.

### What is a network ACL?

A network ACL is a stateless firewall applied at the subnet level.
